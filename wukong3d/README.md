# 黑神话·齐天 3D · Wukong Arena 3D

**真·3D（WebGL）** 的黑暗东方奇幻动作游戏：扮演**孙悟空**（猴相角色：棕毛、猴脸、双耳、金箍头环 + 翎子、尾巴、如意金箍棒），于血月之下迎战群妖与妖王。

> Three.js 已**随仓库本地化**（`three.min.js` 与本页同目录），线上线下均可运行、依旧自包含。

## 如何游玩

- 在线：打开 `index.html`（GitHub Pages 上直接访问该目录）。
- 本地：因为用到 WebGL + 同目录脚本，**建议用本地静态服务器**打开（部分浏览器对 `file://` 加载同级 `.js` 有限制）：
  ```
  cd 到仓库根目录
  python3 -m http.server
  # 浏览器访问 http://localhost:8000/wukong3d/
  ```

## 操作

| 操作 | 按键 |
| --- | --- |
| 移动 | `W A S D` / 方向键 |
| 跳跃 | `空格` |
| 金箍棒（轻击连段） | `J` |
| 重击（击飞） | `K` |
| 闪身（短暂无敌） | `L` / `Shift` |
| 法术·棒扫千军（耗气） | `I` |
| 暂停 | `P` |

移动端自动显示虚拟方向键与攻击键。

## 特色

- **真 3D 角色与场景**：低多边形孙悟空 + 程序化骨架动画（行走 / 挥棒 / 受击），3D 妖兵、妖王。
- **猴相设计**：棕色毛发、浅色猴脸与吻部、双耳、金箍头环 + 两根红色翎子、卷曲尾巴、金色如意棒。
- **氛围渲染**：方向光 + 实时阴影、体积雾、ACES 色调映射、血月与光晕、远山剪影、余烬粒子。
- **打击感**：命中顿帧、镜头震动、击杀慢动作、3D 伤害飘字、火花爆裂、棒光弧扫与法术冲击波。
- **玩法**：三波群妖（狼妖 / 力士）后妖王·玄狼降临，半血进入**狂暴二阶段**（冲锋 / 震地波，需起跳躲避）。

## 技术

Three.js (r128, 本地) + WebGL，程序化几何角色、实时光影、粒子系统；WebAudio 程序化音效。无任何运行时外网依赖。

> 说明：浏览器无法复刻《黑神话·悟空》的 UE5 3A 实时 3D 写实画质，这里是**风格化低多边形 + 氛围光影**的可行方案。
