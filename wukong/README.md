# 黑神话·齐天 · Wukong Arena

一款**黑暗东方奇幻风格的 2D 动作游戏**：扮演齐天大圣，于血月之下持如意金箍棒迎战群妖与妖王。**纯单文件、零依赖**，浏览器打开即玩。

> 说明：这不是、也无法是《黑神话·悟空》那种 UE5 实时 3D 3A 画质——浏览器单文件 Canvas 做不到。这里走的是**剪影描金 + 氛围光影 + 打击感**的现代风格化路线，是网页里最接近那种“精致高级感”的可行方案。

## 如何游玩

直接用浏览器打开 [`index.html`](./index.html)，无需安装、无需联网。

## 操作

| 操作 | 按键 |
| --- | --- |
| 移动 | `A / D` 或 ← → |
| 跳跃 | `W` / `空格` |
| 金箍棒（轻击连段） | `J` |
| 重击（击飞） | `K` |
| 闪身（短暂无敌） | `L` / `Shift` |
| 法术·棒扫千军（耗气） | `I` |
| 暂停 | `P` |

手机 / 平板端会自动显示虚拟按键。

## 玩法与特色

- **连击系统**：轻击三连段 + 重击击飞，命中积累“气”，攒满可放范围法术。
- **闪身无敌帧**：精准闪避妖怪的红光预警攻击。
- **波次 + 妖王**：连续三波群妖后，妖王·玄狼降临，半血进入**狂暴二阶段**（冲锋 / 震地波，需起跳躲避）。
- **打击反馈**：命中顿帧、屏幕震动、击杀慢动作、伤害飘字、火花与魂火粒子。
- **氛围美术**：血月、三层视差远山、体积雾、飘落余烬、暖冷分离调色与暗角；角色为**剪影 + 描金边光**，金箍棒带辉光拖尾。

## 技术

HTML5 Canvas 2D（叠加混合发光 / 程序化角色骨架动画 / 粒子系统）+ WebAudio 程序化音效，全部图形以代码绘制，无任何外部资源。
