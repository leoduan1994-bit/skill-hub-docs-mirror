# 赤色要塞 高清版 · Red Fortress HD

经典 FC《赤色要塞》(Jackal) 风格的俯视吉普车射击小游戏。**纯单文件、零依赖**，本地双击即玩。

## 如何游玩

直接用浏览器打开 [`index.html`](./index.html) 即可，无需安装任何东西、无需联网。

```
# 任选其一
双击 index.html
# 或本地起个静态服务器
python3 -m http.server   # 然后访问 http://localhost:8000/red-fortress/
```

## 操作

| 操作 | 按键 |
| --- | --- |
| 移动 | `W A S D` / 方向键 / 屏幕摇杆 |
| 机枪（扫射步兵） | `J` 或 `空格` |
| 炮弹（轰远处坦克/炮塔/碉堡） | `K` 或 `Shift` |
| 暂停 | `P` |

手机/平板端会自动显示虚拟摇杆与开火按钮。

## 玩法

- 驾驶吉普车向北推进，自动滚动地图。
- **机枪**打前方的敌方步兵；**炮弹**为抛射武器，飞到远处爆炸，可范围摧毁坦克、炮塔和碉堡。
- 碰到 **战俘（白旗）** 即可解救，奖励 500 分。
- 拾取绿色十字补给箱恢复装甲。
- 装甲耗尽损失一条命，3 条命用完游戏结束。每过一关敌人更密、速度更快。
- 最高分通过浏览器本地存储保存。

## 技术

HTML5 Canvas 2D 渲染 + WebAudio 程序化音效，全部图形代码绘制，无任何外部资源。
